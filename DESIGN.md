# A1OS UI Design Spec

## DeepSeek-style Interface
- Sliding sidebar with hamburger menu
- Circular gradient logo (A1OS)
- Settings button at bottom of sidebar
- Chat-first main interface
- Mobile-responsive (fits all screens)
- Dark theme (#0b0b12 background)
- Status indicator (online/offline)

## 7 Modules
1. Memory
2. Knowledge
3. Agents
4. Tasks
5. Cluster
6. Consensus
7. System

## Commands
- add: [text]
- show memory
- show agents
- show tasks
- status
- help

## Build Rule
Build exactly this spec. Do not deviate. Ask before making changes.
