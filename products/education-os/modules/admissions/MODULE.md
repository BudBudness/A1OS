Education OS — Admissions Module
=================================

PURPOSE:
Manage the admission pipeline from application through enrollment.

WORKFLOW:
  application submitted
      ↓
  application reviewed
      ↓
  admission decision
      ↓
  student record created
      ↓
  enrollment confirmed

CORE RECORD:
  - application reference
  - applicant identity
  - guardian information
  - requested class / level
  - application status
  - decision history
  - enrollment link
  - audit history
